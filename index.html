Las 8 áreas de ARES

1. Cuerpo: entrenamiento o movimiento diario.
2. Azúcar: cero azúcar añadida. Fruta sí; refresco, pan dulce, galletas y postres no.
3. Pureza: cero contenido sexual, cero stalkear, celular fuera de la cama.
4. Dios: oración, Biblia y revisión honesta cada día.
5. Inglés: 30 minutos diarios.
6. Dinero: registrar gastos, ahorrar y hacer una acción útil.
7. Orden: cuarto, ropa, cartera, llaves y cosas listas.
8. Puntualidad: llegar al trabajo 6:50 a.m. o antes.

Regla contra recaídas

Nunca fallar dos veces seguidas. Si caes, corriges el sistema y vuelves al camino al siguiente acto.

Almacenamiento local

Esta app ya guarda tus datos en este navegador. No debes configurar nada para eso. Para respaldo, exporta JSON o CSV.

Sincronizar con GitHub

No pegues tu token en el código ni lo subas al repo. El token se usa solo en tu navegador. Para mayor seguridad, usa un token fino con permisos mínimos de contenido del repositorio.
No se guarda en el archivo. Puedes elegir guardarlo solo en esta sesión.

Cómo debe quedar en GitHub

Tu sitio usa index.html para la página.
Tus métricas pueden guardarse en data/ares-datos.json.
Cada vez que presiones “Subir datos a GitHub”, la app actualiza ese JSON con un commit.
